Smart Hospitals:

Smart hospitals use sensors, hardware and software to ensure that patients, and staff are monitored for their security. Sensors and hardware will be used to tag patients with id tags, monitor patient vitals, provide a controlled environment, track usage of medication etc. Smart software will be used to offer intra communication among patients, physicians, and family members. and administration. A smart hospital reduces duplication of information. Such an infra structure will allow the hospitals to function efficiently and effectively.

Upon completing this project the student will be able to Describe what a smart hospital is Identify the significant components such as temperature sensors, light sensors, rfid tags, etc. required to design a smart hospital Analyze requirements to construct each component of the smart farm Describe what you want the software to do Implement the requirements and build the smart smart hospital

Smart Farming:

A smart farm uses several sensors to monitor several elements related to the health of a plant. These include light, moisture, nutrients, etc. The smart farm allows the owner of the farm to monitor all these elements related to his plants from anywhere at any time using the Internet.

Upon completing this project the student will be able to Describe what a smart smart farm is Identify the significant components such as humidity sensors, light sensors etc. required to design a smart farm Analyze requirements to construct each component of the smart farm Implement the requirements and build the smart farm.

Smart Grid:

A smart power system is able to get data from various power installations, such as Hospitals, Residential areas, Markets, etc., and monitor the power consumption at each installation. In case of an emergency caused by a power loss at any one installation, the system is able to reroute power from other installations, depending on priorities.

Upon completing this project the student will be able to: Describe what a smart power system is and its capabilities Identify the significant components required to design a smart power system Analyze requirements to construct each component of the smart power system Implement the requirements and build the smart power system

Smart Parking Lot:

A smart parking lot is one that uses sensors and techniques to keep track of which slots have been occupied and which slots remain open. Such a system lets the driver know were he or she may find a spot to park without having to search for open slots. Use of cameras in parking lots can also capture information about cars parked and in case of criminal activities detect the owner of the car using the camera feed.

Additionally, since this information can be connected to the Internet, the owner will also be able to know which time slots tend to be busy and require more monitoring.

Upon completing this project the student will be able to Describe what a smart parking lot is Identify the significant components required to design a smart parking lot Analyze requirements to construct each component of the smart parking lot Implement the requirements and build the smart parking lot.

Smart City:

What is a smart city? Typically a smart city is one where resources are used in a smart way to increase efficiency and improve daily life of the citizens living in that city. So, as an example think of : a. using your cell phone to switch on and off lights in your house, b. or locking your bike if you forget to lock it in the parking lot, c. monitor and water your plants, and so on. All these tasks ensure that resources are used efficiently when needed and not used when not necessary. In this semester you will build a smart city by installing smart parking lots smart farms smart inventory systems smart parks smart Traffic lights etc. Each one of these installations require skills from multiple disciplines such as math, science, programming and above all analytical and critical thinking. We hope that you will enjoy this activity, learn new technologies and skills and have fun completing the activities. .
